# NewLaravel

Ambiente básico para criação de novos projetos com o framework Laravel já pre-
configurado.

Ambiente já configurado e instalado com:

* Laravel Pint;
* PHPMyAdmin;
* Sail SSL;
* Tradução do Laravel;
* Livewire; 
* LaraStan;
* DebugBar;
* LaraDumps;
* AlpineJS;
* Tailwind;
* DaisyUI;
* MaryUI;
* Script Sail personalizado.
